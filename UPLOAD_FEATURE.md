3. **路由更新**

   - 每页图片上传与文字编辑
   - 表单验证
   - 上传进度反馈

2. **类型定义**
   - `CreatePictureBookRequest` 接口

3. **API 工具**
   - `createPictureBook()` - 创建绘本

4. **路由更新**
   - 添加 `/upload` 路由
   - 需要登录认证
   - 页面标题设置

5. **导航栏更新**
   - 添加"上传绘本"链接（仅登录用户可见）
   - 桌面和移动端都支持

## 🎯 使用指南

### 1. 访问上传页面

登录后，在导航栏中点击"上传绘本"，或直接访问：
```
http://localhost:5173/upload
```

### 2. 上传封面

- 点击封面区域或拖拽图片
- 支持格式：PNG, JPG, GIF, WebP
- 最大大小：5MB
- 自动预览上传的图片

### 3. 填写基本信息

**必填字段**：
- 标题：绘本名称
- 作者：作者姓名
- 适用年龄：0-3岁 / 3-6岁 / 6-9岁
- 主题：动物 / 自然 / 成长 / 冒险 / 友谊 / 科普 / 传统文化
- 描述：绘本简介

**选填字段**：
- 插画师：插画作者姓名

### 4. 添加页面

1. 点击"+ 添加页面"按钮
2. 为每页上传图片
3. 填写页面文字内容
4. 可以删除不需要的页面

### 5. 提交发布

- 点击"发布绘本"按钮
- 等待上传完成
- 成功后自动跳转到浏览页面

## 🔧 技术细节

### 多文件上传实现

**后端 (Multer 配置)**:
```javascript
upload.fields([
  { name: 'coverImage', maxCount: 1 },
  { name: 'pages', maxCount: 50 }
])
```

**前端 (FormData 构造)**:
```typescript
const formData = new FormData();
formData.append('title', title);
formData.append('coverImage', coverImageFile);
formData.append('pages', JSON.stringify(pagesData));
pages.value.forEach((page) => {
  if (page.imageFile) {
    formData.append('pages', page.imageFile);
  }
});
```

### 图片存储
- 存储位置：`server/uploads/`
- 文件命名：`{fieldname}-{timestamp}-{random}.{ext}`
- 访问路径：`/uploads/{filename}`

### 数据关联
- 封面图片：`req.files['coverImage'][0]`
- 页面图片：`req.files['pages'][i]`
- 按顺序关联到对应的页面数据
